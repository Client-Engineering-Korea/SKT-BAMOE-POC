### 2.3 IF-THEN과 DB조회를 모두 포함하는 Rule

#### 2.3.1 정의
**IF-THEN + DB조회 Rule**은 다음 패턴 중 하나에 해당하는 경우입니다:
1. IF 조건문 내부에 DB조회가 있는 경우
2. DB조회 직후 그 결과를 IF 조건에서 사용하는 경우
3. IF 조건 → DB조회 → NESTED IF 패턴
