**1. 판매 기간 관리**
- 판매시작일, 판매종료일 정확히 설정

**2. 재고 동기화**
- 실시간 재고 반영

**3. 판매 가능 여부 플래그**
- 수동 판매 중지 기능

- 재고와 별개로 관리

**4. NULL 처리**
- 판매시작일, 판매종료일 NULL 처리
- NULL인 경우 제한 없음으로 간주

**5. 재고 관리 대상**
- 물리적 상품: 재고관리 = "Y"
- 서비스 상품: 재고관리 = "N"

**6. 오류 vs 성공 반환**
- 상품ID NULL: 실패 반환
