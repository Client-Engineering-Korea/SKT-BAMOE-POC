### 3.2 계산위약금 (위약금 계산)

**설명**: 경과 개월수에 따라 위약금을 계산

**입력 파라미터**:
- 약정정보
- 경과개월수

**주요 처리 로직**:

1. **경과 개월수별 위약금 산정** (line 53-59)
   - IF 경과개월수 < 12
     - 위약금 ← 약정정보.기본위약금 (100%)
   - ELSE IF 경과개월수 < 24
     - 위약금 ← 약정정보.기본위약금 × 0.5 (50%)
   - ELSE
     - 위약금 ← 0 (면제)

**반환값**: 위약금

**위약금 산정 기준**:
- 0~11개월: 기본위약금 100%
- 12~23개월: 기본위약금 50%
- 24개월 이상: 위약금 없음
